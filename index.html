Connect a Google Sheet

Every line comes from your sheet. Paste a Publish-to-web CSV link or a normal share link.

  1. File → Share → Publish to web, pick the tab, format Comma-separated values (.csv), Publish.
  2. Paste that link above. A plain /edit link works too if the sheet is link-shared.
  3. Sections are divider rows (EXPERIENCE, PROJECTS…); the row under each divider is a decorative label and is skipped.
  4. Fixed columns: Title, Tags, Presets, Date, Bullet 1–4, Last Modified, Notes.
